.. _启动前准备:

启动前准备
+++++++++++

在调试机器人之前，应验证以下项目：

- 验证紧急停止装置和操作员防护装置。
- 根据相关标准，设计防护装置以确保机器人在不偏离预定路径的情况下停止（停止类别1）。
- 缓慢移动机器人超出预设工作空间的范围，以验证运动是否受限。
- 使机器人运动超出最大和最小角度，以验证运动是否受限。

.. _启动机器人:

启动机器人
+++++++++++

机器人首次运行时，可能因接线错误、安装紧固不当或参数设置不匹配导致机器人做出预期之外的动作。

.. warning:: 

   **设备意外运行风险：**

    - 确认机器人已正确、牢固安装。
    - 采取必要措施，确保机器人运动部件不会发生非预期运动。
    - 确认急停装置功能正常且位于操作区域可触及范围内。
    - 启动系统前，确认运动路径无障碍物且系统已就绪。
    - 首次启动测试应以低速运行。

   未遵循上述要求可能导致人员死亡、重伤或设备损坏。

若机器人电源因断电或故障意外中断，机器人将无法以受控方式减速停止。

.. warning:: 

  **设备意外运行风险：**

    - 需确保制动器失效状态下的运动不会造成人员伤害或设备损坏。
  
  未遵循上述要求可能导致人员死亡、重伤或设备损坏。 

.. warning:: 

  **高温表面风险：**

  - 禁止在高温表面附近放置易燃或热敏感部件。
  - 通过满负荷条件下的试运行，验证系统散热性能是否符合要求。
   
  未遵循上述要求可能导致人员死亡、重伤或设备损坏。

**启动机器人流程如下：**

- 打开移动底盘左侧的电池仓，拨动电源开关，使其处于 **I** 位置。
- 按下升降机构背面的电源按钮，整机指示灯青色呼吸，此时，系统上电完成。
- 打开Coboπ软件，连接机器人。
- 在首页点击 **电源** 按钮，机械臂上电。
- 点击 **使能** 按钮，分别给左右机械臂上使能。
- 点击 **设置** > **外部轴设置** >, 点击 lift，waist，head0和head1后的 **使能** 按钮，分别给头、颈、腰和升降机构上使能。

**关闭机器人流程如下：**

- 点击 **设置** > **外部轴设置** >, 点击 lift，waist，head0和head1后的 **使能** 按钮，分别给头、颈、腰和升降机构下使能。
- 点击 **使能** 按钮，分别给左右机械臂下使能。
- 在首页点击 **电源** 按钮，机械臂下电。
- 长按升降机构背面的电源按钮15秒，整机指示熄灭，此时，系统断电。
- 打开移动底盘左侧的电池仓，拨动电源开关，使其处于 **O** 位置。

.. Attention:: 

    非紧急状态下，必须严格遵循标准启停流程操作机器人系统。严禁强制断电，否则将导致数据丢失及系统可靠性风险。

.. _操作机器人:

操作机器人
+++++++++++

机器人有两种工作模式：手动模式、自主模式。

.. _手动模式:

手动模式
----------

有以下机种方式手动操作机器人：

- 手柄控制（参考 :ref:`无线手柄`）
- JAKA Coboπ软件（参考软件用户手册）
- SDK API

.. warning:: 

   在手动模式下控制机器人时，操作员有责任确保附近人员的安全。未意识到风险和未小心操作机器人可能导致人员受伤或设备损坏。

   - 在手动模式下控制机器人时，小心以避免与任何人员或物体发生碰撞。
   - 在未清楚看到机器人的情况下，避免手动控制机器人。
   - 您可以将机器人移动至地图上的禁止区域和非首选区域中。
   - 即使在您未禁用保护区域的情况下，您也可让机器人与障碍物发生碰撞。如果您禁用保护区域，则机器人不会在检测到障碍物后停止。

.. _自动模式:

自动模式
----------

在此模式下，机器人执行预先编程好的任务。有以下几种方式实现机器人自主运动：

- JAKA Coboπ软件（参考软件用户手册）
- SDK API
