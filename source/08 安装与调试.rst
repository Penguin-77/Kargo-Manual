安装与调试
############

在阅读本章前，请确保您已经详细阅读并充分理解 :ref:`安全规范` 。

本章将快速介绍JAKA机器人的基本组件与使用方法，作为对机器人的初步了解。详细的机械电气规格请参考其它章节。

.. _运输及保存:

运输及保存
************

.. _运输条件:

运输条件
+++++++++

机器人运输时需要使用原包装，因此如果有运输需求，请务必保存好原包装。

机器人吊装时，应采取相应措施进行定位，避免产生意外运动造成损伤。

.. WARNING:: 

    - 确保抬升设备时作业人员背部或身体其他部位不会过分负重。使用适当的抬升设备。JAKA不对设备运输过程中产生的损害负责。
    - 请遵守各地区及国家有关搬运法规。
    - 确保安装机器人时严格遵守安装指示。

.. attention:: 

    如果运输机器人时未使用其原始包装，所有保修都将失效。

.. _保存条件:

保存条件
+++++++++

1.	保存温度：-10~50℃：
长期保存时，为了维持其可靠性，建议将温度维持在25±10℃内。请避免急剧的温度变化（10℃/h以上）。

2.	保存湿度：20~85%RH
长期保存时，为了维持其可靠性，建议将湿度维持在45~65%内。保存时避免结露或发霉。

3.	防静电
保存在极端干燥的状况下容易产生静电，静电放电时的冲击可能破坏半导体。请放入防静电袋中保存。

4.	其他环境条件
请保存在不会产生有毒气体、污垢、尘埃较少的环境中。保存期间请勿在其上面放置重物。

.. _拆包及搬运:

拆包及搬运
***********

Kargo机器人采用纸箱加托盘的包装形式。操作步骤如下：

1. 拆开并取下外层纸箱。
2. 将放置于泡棉内的斜坡木板取出，并将其放置于托盘前并将二者对齐。
3. 取出电源线、手柄等配件。
4. 取下固定泡棉。
5. 打开移动底盘的电池仓门，拨动电源开关，并按下位于升降机构背面的电源按钮，此时指示灯青色呼吸，表示机器人已上电。
6. 长按位于升降机构背面的释放抱闸按钮3秒，待指示灯变为蓝色后，表示抱闸已释放。
7. 至少由两人推动机器人，直至离开托盘和斜坡。

.. attention:: 

    在推动机器人前，务必确保托盘及斜坡木板已可靠固定，以防止机器人倾覆。

.. _末端执行器安装:

末端执行器安装
***************

.. _安装前准备:

安装前准备
+++++++++++

为避免机器人运行异常，请遵循以下要求：

- 在操作机器人之前，验证负载是否符合机器人所规定的负载范围。
- 根据机器人的最大负载能力，限制最大负载。

.. _安装步骤:

安装步骤
++++++++++

末端执行器安装步骤如下：

1. 将末端执行器固定到机械臂工具法兰安装孔位上 ( **1** )

 - 节圆直径符合DIN ISO 9409-1标准，为50 mm（1.97 in）：4个M6 螺纹孔（ **2** ），拧紧扭矩：15.3 Nm（135.42 Ibf·in），螺钉性能等级不低于12.9。
 - 节圆直径为50 mm（1.97 in）：1 个安装孔直径为6 mm（0.24 in），公差等级 H7 ( **3** )
   
   .. figure:: ./images/末端法兰.*
    :align: center
    :width: 5cm

    末端法兰

.. raw:: html

   <div style="clear: both;"></div>

2. 机械臂末端法兰尺寸信息参考 :ref:`机械臂法兰尺寸` 。
3. 在Coboπ中设置负载，设置负载步骤参考Coboπ软件用户手册。

.. note:: 

    - 请严格遵守影响机器人负载能力的重量及距离限制。
    - 根据ISO9409-1：2004，定位销孔的中心应与机械接口坐标系(GB/T16977-2019)+Xm的轴线对齐，JAKA机器人末端执行器定位销孔的中心在此基础上顺时针偏移45°。
